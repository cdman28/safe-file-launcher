# 공유파일 보호 열기 1.0

> **Summary**: 공유 폴더의 파일을 실수로 수정/삭제하지 않도록, 안전하게 복사 후 열어주는 Windows 데스크톱 앱

> **Last Modified**: 2026-02-27

## 개요

네트워크 공유 폴더(NAS 등)에 있는 자주 사용하는 파일을 등록해 놓고,
클릭 한 번으로 **지정된 작업 폴더로 복사한 뒤 열어주는** 앱입니다.

원본 파일은 절대 수정되지 않으며, 항상 복사본으로만 작업하게 됩니다.
같은 파일을 다시 열면 원본에서 최신 내용으로 **덮어쓰기**하여 복사합니다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| 📂 파일 등록 | 파일 선택 다이얼로그 또는 드래그앤드롭으로 등록 |
| 🔒 보호 열기 | 원본을 작업 폴더로 복사 → 복사본을 열기 (항상 최신 내용) |
| 📁 작업 폴더 설정 | 파일이 복사될 안전한 로컬 폴더 지정 |
| 🎨 파일별 색상 구분 | 카드 상단 색상 바로 파일 시각적 구분 (10가지 색상) |
| 🕐 열기 이력 | 최근 50건까지 열기 기록 확인 |
| ✋ 드래그앤드롭 등록 | 로컬 및 NAS 네트워크 파일 모두 끌어다 놓으면 바로 등록 |

## 설치 및 실행

### 설치형 (권장)
`공유파일 보호 열기 1.0 Setup.exe`를 더블클릭하면 자동 설치됩니다.
- 관리자 권한 불필요
- 바탕화면/시작메뉴 바로가기 자동 생성
- 작업표시줄 고정 지원

### 개발 모드
```bash
npm install
npm start
```

### 빌드
```bash
npm run build:win
```

## 사용법

1. **작업 폴더 설정**: 우측 상단 ⚙️ 클릭 → 폴더 선택 → 저장
2. **파일 등록**: "파일 선택하여 등록" 클릭 또는 파일 드래그앤드롭 (NAS 포함)
3. **파일 열기**: 카드의 "복사 후 열기" 클릭 → 자동 복사 & 열기
4. **이력 확인**: 우측 상단 🕐 클릭

## 동작 방식 (복사 정책)

- 같은 파일을 반복해서 열면 **항상 원본에서 최신 내용으로 덮어쓰기** 복사합니다.
- 파일이 다른 프로그램에 의해 잠겨 있어 삭제가 불가능한 경우에만 `파일명 (1).xlsx` 형식으로 별도 복사됩니다.
